许多使用scss的插件由于不了解所以也没有使用，放弃了一些可能很适合项目的插件，很可惜
在插件的引用方面更加熟练了，了解了成熟框架和ui在项目中的重要作用，在jquey的使用上有一点进步，但是感觉有一些css常见的用法比如通过transform实现
字元素的居中效果这部分是查了之后才用的，没能很熟练的直接运用。需要复习一下css方面的知识点
